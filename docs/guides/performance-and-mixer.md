# Performance & mixer

## Layout (top → bottom)

1. **Top bar** — brand, Performance / Library, **MST · CUE · PHN**, Audio, Help, Settings  
2. **Waveform well** — scrolling detail for both decks  
3. **Deck A · Mixer · Deck B**  
4. **Library strip** — same tracks as Library mode (smaller)

## Mixer (center)

Left → right:

**GAIN A · blue kill LEDs A · EQ + fader A · HI/MID/LOW labels · EQ + fader B · LEDs B · GAIN B**

- **GAIN** — trim / input level per channel  
- **HI / MID / LOW** — EQ knobs; click the **blue LED** to kill that band  
- **Faders** — channel volume; **VU** meters sit on the **outside** of each fader  
- No crossfader in v1 (by design)

## Deck panel

- Title / artist, **headphones (PFL)**  
- Big **BPM**, pitch %, Camelot key, remaining time  
- Overview waveform + **pitch strip**  
- **Play · CUE · SYNC · FILTER · AMT · FLANGER · WET · LOAD**

FILTER and FLANGER are **pads** (on/off). **AMT** and **WET** set how much.

## Loading while playing

You **cannot** load into a playing deck. Pause first. LOAD shows a lock and a short toast if you try.
