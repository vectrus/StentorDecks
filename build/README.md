# Build resources

| File | Use |
|---|---|
| `icon.png` | Brand raster (1024²). Source: `brand/stentordeck-icon.png`. Copied to `extraResources` for splash / BrowserWindow icon. |
| `icon.ico` | Windows exe + NSIS + Desktop shortcut icon. Generated by `npm run icons` (`scripts/make-windows-icon.mjs`). |

Copy updates from `brand/` → `build/icon.png`, then `npm run icons`, before packaging if the mark changes.

**Note:** `win.signAndEditExecutable` stays `false` so packaging does not unpack `winCodeSign` (darwin symlinks need Admin/Developer Mode on Windows). Icon + version metadata are embedded by `scripts/embed-win-icon.cjs` (`afterPack` + `rcedit`).

## Shipping a double-clickable app (no command window)

```bash
npm run dist          # NSIS installer → Desktop + Start Menu shortcuts
# or
npm run dist:dir      # unpacked exe under release/win-unpacked/
npm run shortcut      # Desktop .lnk → that exe
```

Explorer: double-click **StentorDeck** shortcut / `StentorDeck.exe`. Closing the window runs graceful shutdown (analysis, DB, IPC).
